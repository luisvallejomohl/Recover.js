Recover.js

There are many, many, many obsolete javaScript features, such as Number.toInteger, ParallelArray, Object.observe, Function.arity and more. More features will be obsoleted in the future, such as Object.prototype.__defineGetter__, Object.prototype.__proto__ and many more! To solve this problem, we have brought you Recover.js. You can recover old functionality:
var words = [
'foo',
'bar',
'hello,
'world',
];
var word = words[Number.toInteger(Madoth.random() * 4)];
console.log(word);
was once valid javaScript, but no longer is (unless you install Recover.js!) If you install Recover.js, the code above will be vaild again!

You can get the current release at https://recover.js.org/Recover.js, or view all releases on Github.
