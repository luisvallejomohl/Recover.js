Recover.js

There are many, many, many obsolete javaScript features, such as Number.toInteger, ParallelArray, Object.observe, Function.arity and more. More features will be obsoleted in the future, such as Object.prototype.__defineGetter__, Object.prototype.__proto__ and many more! To solve this problem, we have brought you Recover.js. You can recover old functionality: var words = [ 'foo', 'bar', 'hello, 'world', ]; document.querySelector('#generateRandomWord').addEventListener('click', function(){ var word = words[Number.toInteger(Math.random() * 4)]; document.querySelector('.wordContainer').innerHTML = word; }); was once valid javaScript, but no longer is. That is no longer the case: If you install Recover.js, the code above will be vaild again!

You can get the current release at https://recover.js.org/Recover.js, or view all releases on Github.
